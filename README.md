# Gulp Setting

## Install Plugins

```
$ node -v
$ npm init -y
# $ npm install gulp -g
$ npm install gulp --save-dev
$ npm install [pligins] --save-dev

# $ npm install --save-dev @babel/preset-env
```

## Set gulp-slim
[plugin](https://www.npmjs.com/package/gulp-slim)

```bash
$ sudo gem install slim -v '>= 3.0.2'
$ gem update slim
$ npm install gulp-slim --save-dev
```
